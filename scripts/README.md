# Scripts

This directory contains helper scripts that used to live in the repository root.

## Layout

- `scripts/demo/`
  - `demo.sh`
  - `demo-comprehensive.sh`
- `scripts/perf/`
  - `performance-benchmark.sh`
  - `performance-test.sh`
- `scripts/deploy/`
  - `production-deploy.sh`
- `scripts/security/`
  - `security-test.sh`
- `scripts/test/`
  - `test-all-commands.sh`

## Usage

Most scripts are intended to be executed from the repository root, for example:

```bash
./scripts/deploy/production-deploy.sh
./scripts/perf/performance-benchmark.sh
```
